# Board
## Description
mobx를 익히기 위해 만든 간단한 게시판 프로젝트입니다.

**기능 소개**
* 로그인 / 회원가입
* 게시글 작성 / 수정 / 삭제
* 댓글 작성 / 수정 / 삭제
* 검색
